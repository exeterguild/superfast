Tips

Responsive

Currently SuperFast has only one breakpoint. This is 700px. SuperFast will assume that all columns have a 100% width on mobile. You can over-ride this using .mobile-two-column and .mobile-three-column. These are the only grid options for mobile view.

Page Width

SuperFast has two modes. Infinite means the block will take up the whole page (or containing element). Wide gives the element a max-width of 83rem.
